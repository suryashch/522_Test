This is a test environment for the class DSCI 522. There is an environment.yml file for dependencies and a Creative Commons License.

Here is the code you might need when committing to the github -->
- git branch <name>
- git switch <name>
- git switch -c <name>

Remember to push to the new branch once committed -->
- git push origin <branch>

- git fetch --prune <branchname> will sync info from the remote about deleted branches.
